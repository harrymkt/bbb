about us

build beyond boundaries is a map building game for visually impaired computer gamers. it is played using your keyboard.

the game consists of many features for map building.

the maps can also be added to challenge maps, or publish them in public maps menu so players can visit.

features

  • various map building features. over 30 options are available in the builder menu.
  • online. the bbb is a 3d online multiplayer game, meaning it can be played by many players around the world.
  • support. support is available on many ways. support tickets, suggestions, bug reports are available options.

contributing

you can contribute to bbb in many ways.

suggestions

you can suggest new features within the game. that is also count in the contributions.

bug reports

you can also report bugs. bugs include but is not limited to: map related, game issues, and other various issues within the game. everything could not be listed here. you can report bugs from within the game.

documentations

want to contribute more? you can write documentations for the game. please look at this page for more info on how you can contribute.
